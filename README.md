# Wappo Game
It is a remastered copy of old Siemens game named Wappo. Game is created with LÖVE.

## Features
* 100 levels
* Level editor
* Place for your ad.

## Requirements
* [LÖVE](https://love2d.org/)

## Instalation
* Download LÖVE from homepage or install it with help of your package manager
* Clone this repository

## Running game
* cd to game folder
* execute `love .`
